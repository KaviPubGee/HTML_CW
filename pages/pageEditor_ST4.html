User Profile

Technical Description

User Profile page was built using semantic HTML elements such as <section>, <h2>, and <p> to provide clear structure and improve readability. CSS was used for layout and styling, including flexbox for alignment and padding/margins for spacing. We used consistent font styles and colors to maintain visual harmony across the site. The use of IDs allowed for internal navigation and section targeting.

Accessibility

Accessibility was addressed by using descriptive alt text for all images, ensuring high contrast between text and background, and keeping the navigation keyboard-friendly. Headings are used in a logical order to support screen readers, and links include meaningful text.

SiteMap

Technical Description

The Site Map page was created using structured HTML lists to represent the navigation paths of the website. An unordered list <ul> with nested list items <li> was used to show the site hierarchy clearly. Semantic tags like <nav> and <section> helped define content roles. CSS was used to style the list with proper spacing and indentation for clarity. Links within the site map point to relevant pages using anchor tags <a> for easy navigation.

Accessibility

To improve accessibility, each link in the site map includes meaningful, descriptive text. The structure uses semantic elements and a clear heading hierarchy for screen reader compatibility. We ensured that users navigating with a keyboard can access all listed links, and contrast ratios were considered for readability.
